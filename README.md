## 技术要点

将学习以下核心概念：
- 样式处理与 shadcn/ui 组件库使用
- 性能优化（图片、链接和字体）
- 文件系统路由
- Supabase 数据库集成
- 数据获取与管理
- 搜索和分页实现
- React Server Actions
- 错误处理
- 表单验证
- Clerk 身份验证
- i18n 多语言实现
- SEO 优化
- Stripe 支付集成
- Google Analytics 数据追踪
- AI 服务集成
  - OpenAI API 调用与流式响应
  - 图像生成与处理
  - 视频生成与处理
- AI 内容的缓存与优化
- 内容审核与安全控制

## 环境要求

- Node.js 18.17.0 或更高版本
- 支持的操作系统：macOS、Windows（包括WSL）或Linux
- GitHub 账号
- Vercel 账号（用于部署）
- Supabase 账号（数据库服务）
- Clerk 账号（身份验证）
- Stripe 账号（支付服务）
- Google Analytics 账号（数据分析）
- OpenAI API 密钥
- Replicate API 密钥
- Runway API 密钥
